### 프로젝트 목표 (완료)
- 반응형 페이지 (web, tab, mob)
  - 미디어 분기점에 반응하는 레이아웃 변화
- 메인페이지 효과 및 이벤트 구현
  - GNB depth2 전체 슬라이드
  - 타이틀배너 반복 애니메이션
  - section1 호버효과
  - section2 ::web:: 리스트 호버시 배경이미지 전환효과(jQ)
  - section2 ::tab::, ::mob:: 페이지 슬라이드 효과(jQ)
  - section3 [<]/[>] 버튼에 반응하는 리스트 전환 효과 (jQ)
- 활용언어 : HTML, CSS, JavaScript(jQuery)
